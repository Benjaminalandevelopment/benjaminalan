Transport Canada - Dynamics 365 Customization

Project Details

  • Client: Transport Canada
  • Software: Figma / Adobe XD
  • Role: UX / UI Designer
  • Timeline: Current
  • Description: Worked with Transport Canada Inspectors and the product owner to customize and optimize Microsoft Dynamics 365 for their business use.

Project Description

The goal of the project was to take Microsoft's Dynamics 365 out of the box and customize the system for specific use by Transport Canada Dangerous Goods inspectors. The UX team worked with product owners to optimize the current layout as well as design custom components for custom features.

Deliverables

  • Wireframes
  • Interactive Prototypes
  • User Testing and Feedback
  • Custom component prototypes
  • Personas
  • Questionnaire Design
  • Mockups for PBI's

Project Goals

To help developers and business build an online application for regulating Transportation of Dangerous Goods Inspections using Microsoft Dynamics.
